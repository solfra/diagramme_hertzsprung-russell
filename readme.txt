Construction d'un Diagramme de Hertzsprung-Russell à partir des données du satellites Hipparcos de l'agence spatiale européenne (ESA)
Les données sont disponibles sur le sites du Centre de Données astronomiques de Strasbourg (CDS) : http://vizier.u-strasbg.fr/viz-bin/VizieR-3?-source=I/239/hip_main

Pour exporter les données, sélectionnez uniquement les colonnes :
     - HIP (H1)
     - Vmag (H5)
     - Plx (H11)
     - B-V (H37)
     - SpType (H76)
Puis dans la colonne préférence sélectionner "max = unlimmited" et format de sortie "XLM+ CSV (astrores)" et ne pas modifier le reste. Cliquez sur Submit et placez le fichier télécharger dans le même répertoire que le code python.

Cette première version n'utilise pas le type spectrale pour la représentation (a venir)
