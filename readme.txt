Constrution d'un Diagramme de Hertzsprung-Russell à partire des données du satélites Hipparcos de l'agence spatiale européenne (ESA)
Les données sont disponibles sur le sites du Centre de Données astronomiques de Strasbourg (CDS) : http://vizier.u-strasbg.fr/viz-bin/VizieR-3?-source=I/239/hip_main

Pour exporter les données, selectionnez uniquement les colonnes :
     - HIP (H1)
     - Vmag (H5)
     - Plx (H11)
     - B-V (H37)
     - SpType (H76)
Puis dans la collones préférence séléctionner "max = unlimmited" et format de sortie "XLM+ CSV (astrores)" et ne pas modifier le reste.

Cette première version n'uttilise pas le type spectrale pour la représentation (a venir)
